# Project Notes Organiser

**Features**
* Organise notes in folders
* Create/edit/delete notes and folders
* Set notes as complete/in-progress
* Search
* Bulk actions (delete, set complete)

**Technologies used**
* NextJS
* Typescript
* Zustand
* React Query
* Tailwind CSS
